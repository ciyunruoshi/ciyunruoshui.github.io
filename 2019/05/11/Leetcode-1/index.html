Leetcode[1] Two Sum

发表于 2019-05-11

目标：

给定数组和target,从数组中找到两个数字之和为指定的target，返回数组下标

  1. 暴力算法

n= s.length;
对每个数组元素i遍历i+1~n-1,看是否又符合的两个元素
时间复杂度：O(n^2)

代码：

lass Solution {
    public int[] twoSum(int[] nums, int target) {
        int[] out = new int[2];
        int n = nums.length;
        for(int i=0;i<n-1;i++){
            for(int j=i+1;j<n;j++){
                if(nums[i]+nums[j]==target){
                    out[0]=i;
                    out[1]=j;
                    return out;
                }
            }
        }
        return out;
    }
}

  1. HashMap算法

这个算法的思想都是一样的，只不过利用了java的hashmap数据结构，因为hashMap的查询和插入元素是比暴力算法快速的。

代码：

class Solution {  
public int[] twoSum(int[] nums, int target) {
    int[] out = new int[2];
    int n = nums.length;
    Map<Integer,Integer> table = new HashMap<>();
    for(int i=0;i<n;i++){
        int comment = target-nums[i];
            if(table.containsKey(comment)){
                return new int[]{table.get(comment),i};
            }else{
                table.put(nums[i],i);
            }
    }

    return out;
 }
}
