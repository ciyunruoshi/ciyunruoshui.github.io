leetcode7 String to Integer (atoi)

发表于 2019-05-13

将字符串转换为整型类型，其中的规则是

  • 第一个字符必须是数字或者是’+’ ‘-‘,不是的话就输出0
  • 数字必须是连续的数字， 类似43a4只输出43而不是434

知识点：

  • Character.isDigit(“char”)返回boolean值，作用就是判断字符类型是不是数字。
  • 字符串的trim()的方法，作用是用于删除字符串的头尾空白符。

代码：

class Solution {
    public int myAtoi(String str) {

        if(str==null)
            return 0;

        str = str.trim();

        if(str.length()==0)
            return 0;
        int sign=1;
        int start=0;
        int n = str.length();
        long sum=0;

        if(str.charAt(0)=='-'){
            sign=-1;
            start++;
        }else if(str.charAt(0)=='+'){
            sign=1;
            start++;
        }

        for(int i=start;i<n;i++){
            if(!Character.isDigit(str.charAt(i)))
                return (int)(sum*sign);
            sum = sum*10+(str.charAt(i)-'0');

            if(sum*sign>=Integer.MAX_VALUE||sum*sign<=Integer.MIN_VALUE){
                System.out.println(sum);
                return sign==1?Integer.MAX_VALUE:Integer.MIN_VALUE;
            }
        }
        return (int)(sign*sum);
    }
}

编写程序的重要点：

  1. 注意极端情况，字符串为”” 和字符串为 null
  2. 整数为 0 或者是 最大最小值
  3. 要判断某个数组不断叠加起来是否超过Integer的最大最小值，不能用int类型去判断，而要用long或者Double等去判断是否超过int型的极限。
