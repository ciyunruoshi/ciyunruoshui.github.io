关系型数据库(1) 概念

发表于 2019-06-01 | 更新于 2019-06-02

###关系型数据库

  1. 关系数据模型建立在表的集合基础上。
  2. 关系的模式是指它的逻辑设计，关系的实例是指它在特定时刻的内容。
  3. 元组是指数据表中的一行数据，有n个属性组成
  4. 关系的超码是一个或多个属性的集合。这些属性上的取值保证可以唯一识别出关系中的元组。候选码就是最小的超码，一半都是主键。

####SQL(结构化查询语言)

  • 基本类型
  • char(n) 固定长度的字符串
  • varchar(n) 可变长度字符串
  • int 整数类型
  • smallint 小整数类型
  • numeric(p,d) 定点数，精度又用户指定
  • real,double precision 浮点数与双精度浮点数
  • float(n) 精度至少为n的浮点数

JDBC是java程序连接数据库服务器的应用程序接口（API）
